INTRODUCTION
------------

This program implements a web based management system for
product management. Once can add, edit, and delete a product.


REQUIREMENTS
------------

-- Java Compiler
-- MySql database(provided in text file)
-- Tomcat 9.0 Runtime Environment


RECOMMENDED MODULES
-------------------

 None


INSTALLATION
------------
 
Java Virtual Machine


CONFIGURATION
-------------
 
Runtime Environment(Tomcat Server)
MySql Server


TROUBLESHOOTING
---------------

Tomcat might not find the classes if 
the paths were not configured correctly.v
Include all classes in one folder, and remove 
the packages in each class.
